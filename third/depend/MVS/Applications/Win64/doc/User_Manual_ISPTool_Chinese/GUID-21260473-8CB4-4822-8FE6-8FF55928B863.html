使用相机参数校正

ISPTool支持从相机导入校正参数。

  • 校正前需连接相机并完成Pipeline配置，且需确保连接相机前已关闭所有ISP效果相关的使能节点。

  • 相机支持的算法模块才可导入相机参数进行校正，即Pipeline配置中未标注仿真的模块。

  1. 在配置PIPELINE处单击需要进行操作的算法模块。
    说明

    Pipeline中的算法模块需按照从左至右的顺序依次操作。

  2. 在图像操作区域左侧单击导入相机参数，如下图所示。
    图 1 导入相机参数

    图像操作区域左侧将显示导入的校正参数，如下图所示。

    图 2 显示相机校正参数
  3. 单击校正参数下方的开始校正，实现对图像的算法校正，如下图所示。上方为输入原图，下方为校正后的结果。
    图 3 相机参数校正结果
  4. 可选操作： 单击校正后图像右上角的可将结果图片保存至本地。
  5. 单击界面左下方的进入下一模块或配置PIPELINE处的下一个算法模块，可进入下一个算法模块校正界面。
    说明

    进入下一模块后，已完成校正的模块会显示。
