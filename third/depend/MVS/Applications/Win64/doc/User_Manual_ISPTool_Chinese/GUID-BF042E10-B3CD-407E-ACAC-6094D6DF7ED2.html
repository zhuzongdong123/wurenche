CCM

CCM是一个3*3的色彩校正矩阵，可调整RGB中红、绿、蓝三色分量的比例关系。

使用CCM模块时可先进行标定再校正，也可省略标定过程，直接导入校正参数进行图像校正，通过点击直接校正实现。

标定参数

CCM标定支持输入多张标定图像和参考图像，对校正前的标定图像和参考图绘制ROI以提取输入颜色和参考颜色。

说明

图像列表最多支持导入24张图像。

ROI绘制

支持24色卡ROI、140色卡ROI、自定义ROI和自由ROI四种绘制方式，具体介绍可参考图像降噪章节。

说明

对于CCM标定，绘制ROI时预览窗口的图像上会显示ROI序号。

使用输入颜色

  • 不使用：无输入颜色。

  • 原始颜色：点击原始颜色并选择需导入的txt文件，或者点击增加行并选择原始颜色进行添加。

  • 参考颜色：点击参考颜色并选择需导入的txt文件，或者点击增加行并选择参考颜色进行添加。

  • 全部颜色：点击原始颜色或参考颜色可分别导入参考颜色的txt文件，或者点击增加行并选择原始颜色和参考颜色进行添加。

说明

  • txt文件记录颜色格式的要求：一行代表一个颜色，按照r,g,b的格式写入，不允许有空白行；原始颜色和参考颜色应分别存在两个文件中。

  • 原始颜色和参考颜色个数限制均为140，且原始颜色与参考颜色个数需保持一致。

  • 输入颜色过程中如需删除，可点击删除行。

Gamma类型

可根据实际需求进行选择，分为以下4种。

None

不启用Gamma。

User Value

通过设置Gamma数值进行控制，相关参数如下：

Gamma值

输入Gamma数值，范围为[0，3.9]。

User Curve

通过导入Gamma曲线文件进行控制，相关参数如下：

Gamma曲线

单击导入曲线，选择本地的Gamma曲线文件进行导入。

说明

Gamma曲线文件请联系技术支持获取。

LRGB2SRGB

由算法内部生成Gamma值。

校正参数

CCM校正可对以下参数进行调整：

Gamma类型

设置方法和标定参数中相同。

量化系数

可将CCM矩阵中的数值整数化，便于进行运算。设置后CCM矩阵中的数值会随之改变。

CCM矩阵

单击编辑可修改矩阵数值，从而调整RGB中红、绿、蓝三色分量的比例关系。

说明

当校正参数中的量化系数选择1时，矩阵数值可输入浮点数。

饱和度使能

使能后，饱和度下的滑块才可拖动，且量化系数和CCM矩阵不可编辑。

饱和度

拖动滑块可调整图像整体饱和度，CCM矩阵中的数值会随之改变。
