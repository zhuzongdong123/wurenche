Gamma

Gamma模块可对图像进行非线性色调映射。

使用Gamma模块时无需标定，点击直接校正获取校正参数。

校正参数

Gamma类型

可根据实际需求进行选择，分为以下5种：

None

不启用Gamma。

User Value

通过设置Gamma数值进行控制，相关参数如下：

Gamma值

输入Gamma数值，范围为[0.1，3.9]。

User Curve

通过导入Gamma曲线文件进行控制，相关参数如下：

Gamma曲线

单击导入曲线，选择本地的Gamma曲线文件进行导入。

说明

Gamma曲线文件请联系技术支持获取。

LRGB2SRGB

由算法内部生成Gamma值。

SRGB2LRGB

由算法内部生成Gamma值。
