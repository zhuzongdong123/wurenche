Add Remote Camera

You can add GigE Vision camera NOT in the same local subnet with the Software to the device list.

  1. Right-click the network interface card (for example, Local Area Connection in the following picture) to open the right-click menu.
    Figure 1 Right-click Menu
  2. Click Add Remote Camera to open the Add Remote Camera window.
    Figure 2 Add Remote Camera
  3. Enter the camera's IP address and then click OK.
