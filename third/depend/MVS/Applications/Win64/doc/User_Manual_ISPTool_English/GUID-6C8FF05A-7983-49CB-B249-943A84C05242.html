Release Notes

Key updates of each version are as follows.

V4.4.0

This version was released on August, 2024.

Updates with the Software new version.

V4.3.0

This version was released on November, 2023.

  • Adds the function of quick correction.

  • Adds the button of hue and saturation in the image operation area.

  • Supports editing imported images.

  • Supports manually drawing a curve line.

  • Updates the parameters of CCM.

  • LUT supports setting the LUT line type.

  • Adds the algorithm of local contrast enhancement.

  • Embeds the ISP tool in the MVS Software.

V1.1.0

This version was released on September, 2021.

  • Updated the running environment, see System Requirements.

  • Added General Settings, Video And Picture Settings, Tool, and Help on the menu bar. See Menu Bar.

  • Added the LSC (Lens Shading Correction) algorithm. See Lens Shading Correction (LSC).

  • Added the DDS algorithm. See DDS.

  • The SixAixs algorithm is renamed as Super Palette. See Super Palette.

  • Added the purple fringe correction algorithm. See Purple Fringe.

  • Updated the parameters of sharpness. See Sharpness.

  • Added the super binning algorithm. See Image Scaling.

This version was released on March, 2021.
