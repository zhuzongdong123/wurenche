Import an Image from Camera

You can import an image from a camera into the Software.

Note

The Tool only supports some camera models.

  1. Click to open the Camera List pane.
    Figure 1 Camera List
  2. Hover the cursor onto a camera.
  3. Click to connect the camera.

    The image from the camera will be displayed.

  4. Click to start acquiring an image from the camera.
  5. Optional: Perform the following operations if needed.
    Option Description

    Capture Image

    Click to capture an image.

    Start Recording

    Click to start recording.

    Zoom Image

    Click / to zoom in/out on the image.

    Display in Actual Size

    Click to display the image in its actual size, and click to fit it to the size of the window.

    Configure Pipeline

    See Configure the Pipeline for details.

    Import Another Image

    Click to import another image to replace the current one.
