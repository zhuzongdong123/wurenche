Add USB3 Vision Camera

You can add a USB3 Vision camera to the Software by automatically enumerating the camera connected to the PC where the Software runs.

After you connect a USB3 Vision camera to the PC on which the Software runs, the camera will be automatically enumerated if the USB driver is properly installed. For details about automatically enumerating cameras, see Automatically Enumerate Local Cameras.

Note

Make sure you have installed the USB 3.0 driver when installing the Software: on the Settings page, check USB 3.0. If you do not install the USB 3.0 driver during installation, the Software cannot enumerate the camera. You can install the Software again if you need to add a USB3 Vision camera.
