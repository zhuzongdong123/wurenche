Import an Image from PC

You can import an image from the PC into the Software.

Make sure that the name of a BMP image contains its gain value, such as "image_g0.bmp", and that the name of a RAW image contains its gain value, width, height, and joint No., such as "image_g0_w1440_h1080_pBayerRG8.raw".

Note

The joint No. is only valid for the Denoise module.

  1. Click on the toolbar.
  2. Select an image.
  3. Import the image into the Software.
  4. Optional: Perform the following operations if needed.
    Option Description

    Zoom Image

    Click / to zoom in/out on the image.

    Display in Actual Size

    Click to display the image in its actual size, and click to fit it to the size of the window.

    Configure Pipeline

    See Configure the Pipeline for details.

    Import Another Image

    Click to import another image to replace the current one.
