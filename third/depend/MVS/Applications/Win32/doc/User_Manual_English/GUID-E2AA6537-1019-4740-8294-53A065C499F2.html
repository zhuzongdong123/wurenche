Release Notes

This section lists key updates of the Software.

  • V4.4.0

  • Updates Before V4.4.0

V4.4.0

V4.4.0 was released on August, 2024. This version optimizes the interface category in the device list and enhances other functions.

  • New Features

    • Supports Minimalist Mode in General Settings. When enabled, the Software pauses automatic refreshing to save system resources.

    • Supports viewing description of feature, including node and sample codes. See Feature Tree.

    • Supports batch connecting and disconnecting multiple cameras. See Acquisition and Live View in Multiple-Window Mode.

    • Supports enumerating serial port devices by specifying Serial Port Id. See General Settings.

    • Supports Bandwidth Detection in Tool, with which you can monitor bandwidth availability at different image acquisition sizes.

    • Supports Mono 14 pixel format.

    • Supports adjusting exposure automatically. See More Functions.

  • Improvements

    • Supports automatically enumerate frame grabbers developed by Hikrobot and cameras connected, under PCIe interface. See PCIe Device Management.

    • Updates Camera Link interface into Serial Port interface, through which you can connect Camera Link cameras. See Serial Port Device Management.

    • Updates Log Viewer Tool, you can view logs from cameras and frame grabbers in one window.

    • Supports using slider controls to select a value for feature values of int and float types.

    • Updates Driver Manager, and now it supports installing and uninstalling PCIe drivers.

    • Modifies RGB and YUV features in live window into Pixel Value, the displayed value depends on the acquired image format. See View Acquisition Status.
