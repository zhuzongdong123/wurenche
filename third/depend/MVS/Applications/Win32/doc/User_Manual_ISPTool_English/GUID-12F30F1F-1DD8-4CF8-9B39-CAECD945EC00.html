Algorithm Introduction

The Software supports correcting an image by the following algorithms: lens shading correction (LSC), denoise, bayer interpolation (demosaic, DDS), color correction (CCM, CLUT, super palette, purple fringe correction), image enhancement (sharpness, LUT, contrast), and image scaling.

Note

The algorithms should be supported by the camera. If an algorithm is not supported by a camera, there will be a blue "Sim" tag in the upper right corner of the algorithm icon as is shown in the figure below.

Figure 1 Pipeline Pane
