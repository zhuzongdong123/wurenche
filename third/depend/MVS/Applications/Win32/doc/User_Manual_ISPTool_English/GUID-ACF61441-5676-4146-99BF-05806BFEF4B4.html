Correct the Image by Setting Parameters Manually

You can correct the image by setting parameters manually if the algorithm module does not support local parameters.

If an algorithm module does not support local parameters, such the CCM, you can click Manually Correct to set parameters manually, and then click Correct to correct the image.

After correction, you can click Next or click the next algorithm module in the Set Pipeline area to correct next algorithm module.
