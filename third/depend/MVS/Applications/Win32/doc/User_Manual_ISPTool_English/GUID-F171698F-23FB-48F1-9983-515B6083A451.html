System Requirements

Make sure the computer where you install and run the Software at least meets the minimum requirements.

Note

The compatibility of online mode is subject to camera models.

Recommended Hardware Requirements

  • CPU: Intel® i3-8100T 3.1 GHz and above

  • RAM: 8 GB and above

  • Graphics Card: Intel® HD Graphics 360

  • Network Adapter: Intel® Pro1000 series

  • USB: USB 3.0 interface

Minimum Hardware Requirements

  • CPU: Intel® E3845 1.91 GHz

  • RAM: 4 GB

  • Graphics Card: 7th Gen Intel® Core Process

  • Network Adapter: Intel® Pro1000 series

  • USB: USB 2.0 interface

Software Requirements

  • Operating System: Microsoft® Windows 7 / 10 (32/64-bit) / 11 (64-bit)

  • SDK Version: Machine Vision Industrial Camera SDK V3.5.2 and later
