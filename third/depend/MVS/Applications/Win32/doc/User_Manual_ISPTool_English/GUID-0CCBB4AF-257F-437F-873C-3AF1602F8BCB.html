Gamma

The Gamma module supports setting nonlinear tone-mapping for images.

Calibration is not necessary for the Gamma module, click Correct to start.

Correction Parameters

User Curve

Gamma Curve

Click Import Curve to import a Gamma curve file from the local PC.

Note

Contact the technical supports to get the Gamma curve file.
