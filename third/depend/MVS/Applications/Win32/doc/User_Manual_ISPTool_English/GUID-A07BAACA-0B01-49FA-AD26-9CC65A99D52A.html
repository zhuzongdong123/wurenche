Preview the Corrected Image

After correcting an image, you can preview its corrected version.

  1. Click to enter the Preview Corrected Image page.

  2. Click Import Original Image.

  3. Select the original version of the corrected image and import it into the Software.

    Note

    • The corrected image will be displayed.

    • If you have opened the original image, you do not need to import it again.

  4. (Optional) Perform the following operations if needed.

    Operation How

    Zoom Image

    Click / to zoom in/out the image.

    Display in Actual Size

    Click to display the image in its actual size, and click to fit it to the size of the window.

    Import Another Image

    Click to import another image to view its corrected version.
