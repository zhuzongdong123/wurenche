Release Notes

Key updates of each version are as follows.

V4.4.0

This version was released on August, 2024.

  • Adds Event Monitor in the main UI for quick event viewing. For details, see Configure the Event.

  • Supports copying node name through keyboard shortcut. For details, see Configure the Event.

  • Adds shortcut entry to Event monitor. See Tool Introduction.

V4.3.0

This version was released on November, 2023.

Added the function of importing/exporting events.

V4.2.0

This version was released on June, 2023.

Provided reference for viewing and configuring frame grabbers' events.
