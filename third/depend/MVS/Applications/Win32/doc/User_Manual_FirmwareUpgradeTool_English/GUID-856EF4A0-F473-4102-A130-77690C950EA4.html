Release Notes

Key updates of each version are as follows.

V4.4.0

This version was released on August, 2024.

  • Changed Camera Link interface to Serial Port and GenTL to PCIe.

  • Added Fuzzy Matching to frame grabber upgrade page. See Upgrade a Frame Grabber.

V4.3.0

This version was released on October, 2023.

Optimized the operations of upgrading and configuring frame grabbers. See Upgrade a Frame Grabber.

V4.2.0

This version was released on June, 2023

You can use the Firmware Upgrade Tool to configure firmware upgrading settings and upgrade the firmware of cameras and frame grabbers.
