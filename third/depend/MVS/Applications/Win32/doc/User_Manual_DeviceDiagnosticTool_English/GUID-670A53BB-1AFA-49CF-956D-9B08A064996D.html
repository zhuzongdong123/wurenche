Release Notes

Key updates of each version are as follows.

V4.4.0

This version was released on August, 2024.

Adds Stop when occur and Timer period type in exception IO locating. See IO WaveForm Detection.

V4.2.0

This version was released on March, 2023

This is the first version of Device Diagnostic Tool. The following are the main features.

  • Supports diagnostic event overview. See Event Overview.

  • Supports detailed event diagnosis. See Event Diagnostic.

  • Supports viewing IO waveforms. See IO WaveForm Detection.

  • Supports log management. See Log Management.
