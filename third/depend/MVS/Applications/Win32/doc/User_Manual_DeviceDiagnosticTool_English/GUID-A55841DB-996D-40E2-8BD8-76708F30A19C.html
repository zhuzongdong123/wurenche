Event Overview

You can take an overview of real-time frame grabber / camera events.

Note

Make sure you have connected frame grabber(s) / camera(s) in advance.

Click Run, and a scoped-map will be generated.

Figure 1 Event Overview

You can right-click and drag the cursor on the scope-map to view the events before/after the current time period.

Click to select a time period in the scope-map to view the diagnostic result of events in 6 seconds. For details, see Event Diagnostic.

Area

Description

X-Axis

Timeline. Unit: min.

Y-Axis

Number of events.

Green Point

Normal events.

Red Point

Error events, such as buffer exception, camera offline, and DDR initialization failed. For details, see Frame Grabber SDK Developer Guide.

Note

The number of error events is displayed in the upper-right corner.

Orange Point

Warning events such as no parameter collected by frame grabber(s) / camera(s).

Note

The number of warning events is displayed in the upper-right corner.

Pause

Click Pause the current running.

Note

The Tool will stop receiving data collected from the device.

Reset

Click Reset to clear the current data and start running.

Select Device

Click Select Device to select frame grabber(s) / camera(s) and click OK.

Note

If you select multiple devices, the Tool will only generate a broken line containing all device events diagnosed.

Lock/Unlock Scope Map

Click to lock the current scope-map and stop displaying the new timeline and event data.

Click to unlock the current scope-map and display the event data received during locking and newly parsed event data.
