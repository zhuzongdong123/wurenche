Release Notes

Key updates of each version are as follows.

V4.4.0

This version was released on August, 2024.

Changed Camera Link interface to Serial Port and GenTL to PCIe.

V4.3.0

This version was released on October, 2023.

Supports importing and exporting parameters of frame grabbers. For details, refer to Import/Export Features.

V4.2.0

This version was released on May, 2023.

Provides the Feature Import/Export Tool to import/export the device parameters, etc.
