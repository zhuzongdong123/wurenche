Tool Introduction

The Feature Import/Export Tool (referred to as "Tool" afterward) can be used for importing and exporting parameters of one or multiple cameras or frame grabbers.

The main interface of the tool is shown below, and you can refer to the table for the related descriptions.

Figure 1 Main Page of Tool
Table 1 Main Page Description

No.

Function

Description

1

Import/Export

Import/Export the parameters of the selected camera(s) or frame grabber(s).

2

Interface Information

Display the interface information on the current PC.

3

Device Information

Display the cameras or frame grabbers and the related information under different interfaces.

4

Device Information Configuration

Set the device information to be displayed in the area 3 as needed. The information can be different when you select different interfaces.
